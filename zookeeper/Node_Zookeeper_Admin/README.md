##WEB使用说明:
在windows环境下直接执行run.bat,打开浏览器访问：http://localhost:3000

##客户端使用说明:
解压下载包中的bin/release_V*.rar,直接打开nw.exe
如应用直接关闭，请检查3000端口是否已被占用

##V1.0版本更新
V1.0版本目前默认连接Zookeeper ip为127.0.0.1:2181端口
